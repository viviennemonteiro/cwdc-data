# CWDC Data Tools

# Lock Up Scraper
This is a PDF parser for lock up lists. 


# Quick entry command line tool
This is vibe coded a command line tool for entering data with simple keyshortcuts in the command line. 

Currently this tool supports entering age, race, and gender which is generally the minimum amount of data we need for most analysis.
